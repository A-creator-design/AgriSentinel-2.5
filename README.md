# 🌾 AgriSentinel 2.5

**Smart Vision, Local Voice, Better Yield**

## 🚀 Overview

AgriSentinel 2.5 is a multimodal agricultural advisory agent that helps farmers detect crop diseases, get treatment suggestions, and access mandi price insights using AI.

## 🔥 Features

* 📸 Image-based plant analysis (Gemini 2.5 Flash)
* 🎤 Multilingual voice output (Telugu, Hindi, English)
* 🌦 Live weather updates
* 💰 Market (Mandi) price insights
* 📡 Designed for low-connectivity environments
* 🌙 Dark UI for outdoor usage

## 🛠 Tech Stack

* Streamlit
* Google Gemini 2.5 Flash
* gTTS
* wttr.in API

## ▶️ Run Locally

pip install -r requirements.txt
streamlit run main.py
